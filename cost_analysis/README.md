# Анализатор расходов

Веб-приложение для анализа личных финансов на Flask.

## Возможности

- ?? Загрузка Excel файлов с расходами
- ?? Хранение данных в SQLite
- ?? Статистика по категориям и месяцам
- ?? Фильтрация по году и месяцу
- ?? Экспорт данных обратно в Excel

## Установка

1. Клонируйте репозиторий
2. Создайте виртуальное окружение:
   python -m venv venv
   source venv/bin/activate
# или venv\Scripts\activate на Windows
“становите зависимости:
bash
pip install -r requirements.txt
запустите приложение:
bash
python app.py
